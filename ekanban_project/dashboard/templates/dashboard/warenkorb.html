Funktionsweise der Berechnung
Verbrauchsberechnung
System berechnet den hypothetischen Kabelverbrauch pro Trommel basierend auf der Depletionsrate (m/Tag). Der Verbrauch wird vom heutigen Datum bis zum ausgewählten Datum summiert.
Bestelllogik
Eine Trommel wird aufgenommen, wenn Lieferzeit + Prognosepuffer größer ist als verbleibende Tage. Prognosepuffer: Defensiv +10 Tage, Neutral +5 Tage, Offensiv +2 Tage. Anschliessend wird der Warenkorb im Demo-Modus bis 500 EUR Warenwert aufgefuellt.
Budget-Constraint
Versandkostenfrei ab 500 EUR Warenwert, sonst +25 EUR Versand. Unter 150 EUR Warenwert kommt ein Zuschlag von +20 EUR hinzu. Nicht-Standardlaengen (100/500/1000 m) erzeugen +20 EUR Schnittkosten pro Position.
Demo-Modus
Beim Klick auf "Jetzt kaufen" werden die bestellten Meter den zugehoerigen Trommeln zugeschlagen. Dadurch werden die Bestände in der Zukunftssimulation erhoeht und koennen spaeter erneut leerlaufen.
